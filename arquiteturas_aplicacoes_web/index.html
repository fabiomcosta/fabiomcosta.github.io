Arquitetura

Koa

  • Framework de servidor para ajudar na criacao de APIs
  • Alternativa mais moderna ao Express
  • https://koajs.com/

NextJS

  • Framework de React
  • Server-side rendering (renderizacao de componentes React no servidor)
  • Roteamento
  • Code splitting
  • https://nextjs.org/

GraphQL

  • GraphQL eh uma query language que possibilita a criação de APIs onde quem faz a requisição (cliente) decide quais informações quer.
  • http://localhost:4000/graphql

Frontend

  • React + Hooks
  • Redux
  • Relay

React + Hooks e Redux

  • Flexiveis
  • Facil de usar
  • Funcionam com qualquer backend
  • Requerem solucao de terceiros para network e cache de dados

Relay

  • Requer backend com suporte a GraphQL
  • Complexo (connections, mutations, fragments ...)
  • Dificil de iniciar, porem muito produtivo apos curva de aprendizado inicial
  • Prove network e cache de dados
